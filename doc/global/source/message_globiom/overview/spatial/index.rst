Spatial process
==============
MESSAGE has global coverage and divides the world into 11 regions (see Figure 1.1 below).

.. image:: /source/message_globiom/overview/spatial/MESSAGE_regions.png
Figure 1.1: Map of 11 MESSAGE regions.
